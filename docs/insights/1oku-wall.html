01. 相談時の課題（Facts）

クラウド型の人事労務システムを提供しているA社。プロダクト自体は優れているものの、市場には大手競合がひしめいており、リスティング広告のクリック単価は年々高騰していました。

広告費を月額100万円から150万円に増額したものの、獲得リード数は比例して伸びず、CPAが35,000円まで悪化。このままでは利益が出ない「詰み」の状態でマケマケにご相談いただきました。

02. 構造の分析とボトルネック

ヒアリングと既存のLPを分析した結果、ボトルネックは「広告の運用スキル」ではなく「市場定義と価値表現の断絶」にあることが判明しました。

当時の断絶ポイント

  • 市場定義の誤り： 大手と同じ「人事部向け」をターゲットにしていたため、資本力で負けていた。
  • 表現の誤認： LPのファーストビューが「効率化」「DX推進」という、どの競合も使っているコモディティ化した言葉になっていた。

03. 売上構造の再設計（Design）

マケマケは戦略顧問として、競合と真正面から戦わないためのポジショニングを再定義しました。

過去の契約データを分析すると、最もLTVが高く導入がスムーズなのは「専任の人事担当者がおらず、社長や総務が兼任している従業員50名未満の企業」であることがわかりました。そこで、ターゲットを人事部ではなく「社長・総務」へ意図的にズラしました。

04. クリエイティブ制作と実行（Execution）

新しい戦略に基づき、マケマケの制作チームがLPを全面リニューアルしました。訴求軸を「DX」から「総務の残業削減・属人化解消」へ変更し、広告の配信キーワードも「人事システム」というビッグワードから、よりニッチな課題解決型キーワードへシフトしました。

05. 改善成果（Results）

新しいLPを公開し、運用を改善した結果、わずか2ヶ月でCPAは35,000円から16,800円へと半減。同予算で獲得できる商談数は約3倍（月間55件）へと急増しました。

「施策（デザインや広告運用）」だけをいじるのではなく、「構造（誰に何を言うか）」から直列化したことで、再現性のある成果を生み出すことができました。
